Title of the project : Sudoku game for GSSOC'22
What the Project Does : This is the simple sudoku game.
How to use: 
1.click on the cell wherever you want to add a number. Once you click on the particular cell, numbers 1-9 will get displayed. select the one you want to add. and that's it.
2.The timer will start once you start the game.
How to install or requirements:
You need the knowledge of HTML, CSS and Javascript.
